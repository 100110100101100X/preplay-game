<h1>Описание</h1>
<p>Рекламная HTML-игра в жанре "hidden objects"</p>
<p>Необходимо найти все предлагаемые объекты, кликнув на каждый из них. При бездействии мыши в течении 5 секунд спрятанный объект подсвечивается.</p>
